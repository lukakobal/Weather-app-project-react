# 🌤️ React Weather App

![React](https://img.shields.io/badge/React-18.2.0-blue?logo=react&logoColor=white) ![OpenWeather](https://img.shields.io/badge/OpenWeather-API-orange)

Enostaven **Weather App** v Reactu, ki omogoča iskanje trenutnega vremena za katerokoli mesto.  
App uporablja **controlled input**, **useState**, in **fetch** za pridobivanje podatkov iz OpenWeather API-ja.

---

## 🚀 Funkcionalnosti

- Iskanje mesta preko inputa ali tipke **Enter**  
- Prikaz trenutne temperature, vlage, vetra in opisa vremena  
- Loading indikator med nalaganjem podatkov  
- Preverjanje napak (če mesto ne obstaja)  
- Input se sam pobriše po uspešnem iskanju  
- Background slika + glassmorphism kartica za lep vizualni efekt  

---

## 💻 Tehnologije

- React 18  
- CSS (glass effect, responsive)  
- OpenWeather API
